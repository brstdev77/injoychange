Synchronizing with character events.

Available for Studio & SitePal

The playback environment generates Event callbacks which correspond to different stages of playback. The site in which the host is embedded can use these Event callbacks to trigger other actions.

On Scene Loaded:
vh_sceneLoaded() - is generated when a Scene is loaded.

On Talk Started:
vh_talkStarted() - is generated when speech begins.

On Talk Ended:
vh_talkEnded() - is generated when speech ends.

Audio Progress:
vh_audioProgress(percentPlayed) - is generated at an audio progress interval.
setStatus(0,1,0) - sets the progressInterval in vh_sceneLoaded
